Beyond the standard dashboard. This is a "Chassis".

ST-CYBER-BASE-CORE is a lightweight UI engine designed to bypass the traditional layout constraints of Streamlit. Inspired by 80s cyberpunk aesthetics and industrial motherboard designs, it provides a "modular socket" system where users can intuitively drag, drop, and snap UI components.

ğŸ› ï¸ Key Features
Modular Snapping: An intuitive UI system where components snap into sockets like hardware chips.

Ultra-Lightweight Core: Stripped of heavy physics or audio processing for maximum performance and easy customization.

Infinite Extensibility: Easily replace the "Unit" content to implement anything from stock market monitors to interactive system diagnostics.
